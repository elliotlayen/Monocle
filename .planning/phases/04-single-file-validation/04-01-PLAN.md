```rust
pub struct FileContent {
    pub content: String,
    pub size: u64,
}

#[tauri::command]
pub async fn read_file_cmd(path: String) -> Result<FileContent, String> {
    tokio::time::timeout(
        Duration::from_secs(30),
        tokio::task::spawn_blocking(move || {
            let metadata = std::fs::metadata(&path)
                .map_err(|e| format!("Failed to read file '{}': {}", path, e))?;
            let size = metadata.len();
            let content = std::fs::read_to_string(&path)
                .map_err(|e| format!("Failed to read file '{}': {}", path, e))?;
            Ok(FileContent { content, size })
        }),
    )
    .await
    .map_err(|_| "File read timed out after 30 seconds".to_string())?
    .map_err(|e| format!("File read task failed: {}", e))?
}
```

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-01 | Tampering | read_file_cmd (file content) | mitigate | encoding_rs replaces malformed bytes with U+FFFD (lossy decoding), never executes content. Content is rendered in read-only Monaco Editor (no eval). |
| T-04-02 | Denial of Service | read_file_cmd (large file) | accept | Existing 30-second timeout on read_file_cmd limits hang time. File size returned in response. No additional DoS vector vs Phase 3. |
| T-04-03 | Information Disclosure | validation results | accept | Validation problems contain line/column/message from user's own local files. No network transmission. Desktop app context. |
| T-04-SC | Tampering | cargo install (encoding_rs, chardetng) | mitigate | Blocking human checkpoint (Task 0) verifies crate legitimacy on crates.io before installation |
</threat_model>
