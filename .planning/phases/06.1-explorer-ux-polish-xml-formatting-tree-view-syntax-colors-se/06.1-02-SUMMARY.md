# Phase 06.1 Plan 02: Segmented Controls, Date Display, Sort, and Filter Summary

Standardized both segmented controls to identical Tailwind+ARIA patterns, wired the existing formatDateFolder utility into tree node rendering, connected dateSortOrder to the actual sort comparator, and added a date filter preset dropdown to the sidebar header.

## Changes Made

### Task 1: Standardize segmented controls
- Added `p-0.5`, `role="group"`, `aria-label="View mode"`, and `aria-pressed` attributes to the file-content-header Tree/Source toggle
- Added `font-medium` class to both segment buttons in file-content-header
- Replaced shadcn `Button` components with raw `button` elements in search-bar to eliminate ghost variant style conflicts
- Aligned search-bar container classes from `bg-muted/50` to `bg-muted` with explicit `h-7` and `border`
- Used `cn()` for conditional class composition in search-bar (previously template literals)
- Removed unused `Button` import from search-bar

### Task 2: Wire formatDateFolder and dateSortOrder
- Imported `formatDateFolder` from `../utils/date-format` in folder-tree-node
- Updated `renderName` to call `formatDateFolder` on non-source directory nodes and display formatted date in parentheses (e.g., "20251223 (Dec 23, 2025)")
- Added `dateSortOrder` to the `useShallow` selector in folder-tree
- Updated sort comparator to detect 8-digit YYYYMMDD folder names via regex and apply sort direction based on `dateSortOrder` state

### Task 3: Add date filter preset dropdown
- Added `DateFilterPreset` type alias (`"all" | "7d" | "30d" | "90d"`) to explorer store
- Added `dateFilterPreset` state field (default: `"all"`) and `setDateFilterPreset` action
- Added Calendar icon dropdown menu to sidebar header using `DropdownMenuCheckboxItem` for preset options
- Calendar button shows `bg-accent` visual indicator when filter is active (not "all")
- Added `filterByDatePreset` callback in folder-tree that filters YYYYMMDD-pattern folders by computed cutoff date
- Non-date-pattern folders and file nodes always pass through the filter
